What is UniVerse?

A next-generation digital platform designed to transform how students learn, collaborate, and engage with their university communities—combining a virtual campus experience, gamified education tools, and a student-driven economy.

  • Students socialize through avatars, customize dorms, and play mini-games in Campus World.
  • Faculty run interactive learning experiences (quizzes, challenges, progress tracking) in Education Mode.
  • Verified students buy/sell/trade safely in a campus-only Marketplace.

Why it matters

Engagement Meet students where they already are—interactive, social environments.
Learning Gamified tools that complement existing coursework and feedback.
Trust Campus-only marketplace + moderation foundations.
Opportunity Open-source collaboration model that builds real portfolios.

Product Pillars

Campus World

A virtual campus hub for social play, mini-games, dorms, and community.

  • Virtual campus hub + hangouts
  • Mini-games hub (quick multiplayer)
  • Dorm customization + progression

Education Mode

Instructor tools to gamify classes with quizzes, flashcards, and assignment quests.

  • Instructor dashboard + class spaces
  • Live quizzes + flashcard battles
  • XP, badges, streaks, leaderboards

Campus Marketplace

A verified student-only economy for buy/sell/trade—designed for trust and safety.

  • Listings + images + categories
  • Messaging + meetup preferences
  • Reputation + reporting

One platform. Multiple surfaces.

UniVerse launches with a unified account + backend, delivered through a Unity mobile client and a web app—web screens can be embedded inside Unity (WebView) for a single-brand feel.

  • Unity Mobile App: Campus World + mini-games + social presence
  • Web App: Marketplace + faculty tools + admin/mod tools
  • Shared backend: identity, economy, education, marketplace

MVP Core Loop

The first launch proves retention with a simple loop that students repeat weekly.

  • Login → choose campus → enter hub → meet friends
  • Play a mini-game → earn coins/XP → decorate dorm → repeat
  • Optional: join a class → play a live quiz → earn learning badges

Roadmap

Phase 1 — MVP

Hub + dorm, 2 mini-games, Marketplace v1, Education live quiz v1, WebView embed.

Phase 2 — Retention

Add games, events, leaderboards, stronger marketplace trust/safety, more education templates.

Phase 3 — Integrations

University SSO and roster imports; official campus events and branded rewards.

Phase 4 — League

Cross-campus matchmaking, seasonal competitions, and partnerships at scale.

Open-source collaboration program

UniVerse is built as an open-source initiative supported by a student collaboration model—contributors gain real-world experience in software development, game design, systems architecture, and teamwork.

  • GitHub-based workflows + mentorship
  • Cross-disciplinary teams (dev, security, design)
  • Portfolio building through production-level features
